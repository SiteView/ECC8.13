attribDelete.java is an ant class that will selectively delete classes in the Dragonflow tree as a component of clean. 

attribDelete.java is a component of anttask.jar (see classes\dev).

To create anttask.jar

1) Compile attribDelete.java (you might need to mount ant.jar)

2) jar cvf anttask.jar COM.ant.attribDelete.class (from \SiteView\classes)
