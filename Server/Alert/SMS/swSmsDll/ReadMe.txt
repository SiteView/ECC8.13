    动态链接库 : swSmsDll 项目概况
========================================================================
汕尾短信发送DLL

通过调用Oracle OCI向数据库发送短信
运行时需要安装Oracle 客户端

编译时需要添加include目录
ORACLE_HOME\oci\include
以及Lib目录
ORACLE_HOME\oci\lib\msvc
(ORACLE_HOME为oracle安装目录）

DLL适用oracle版本为：oracle 9i,oracle 10g以及兼容版本
